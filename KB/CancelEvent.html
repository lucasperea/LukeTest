Cancel an Event

If your event needs to be cancelled for any reason and your organization follow the steps below:

  1. Deactivate the event
    Select Management Menu>>Operations>>Event Management Click the "Inactive" radio button next to the event Click the Update Status button
  2. Notify ticket buyers of the cancellation Select CRM>>Operations>>Email Notification (NEW)
    Select Include next to the cancelled event
    Click the Next button
    Enter the Reply Email Address, Subject, and Email Body - The email should explain that the event has been cancelled and refunds to the patron's credit card will be issued within 5-7 business days. *Uncheck the â€œMarketing Email (Mailing List Members Only). This ensures that the email goes to all patrons that made an order Click the Next button
    Select Plain Template
    Click the Next button
    Enter your email address in the Test E-Mail Address field and click the Send Test button to send yourself a test email to ensure that the email is formatted to your liking When ready to send the email to your patrons, click the Send button
If you're using the Tix Merchant account, please contact your Tix Representative Contact your Account Manager as soon as possible. Your Account Manager will make arrangements to cancel the existing orders, issue refunds, and assess the refund fees due by your organization.

If you're using your own merchant account, please continue to the next steps.

  1. Cancel the existing orders in the Tix system
    Select Reports>>Operations Reports>>Event Detail Report (NEW)
    Under Event Selection select Choose Events and choose the event that will be cancelled
    Under Seat Status select Choose Seat Statuses and select Sold
    Click the Submit button
    Click on the Order Number in the Order Number column to view the Order Details page for an order
    Scroll to the bottom of the Order Details page to the Cancel Order heading (Cancel Order must be enabled on your User Account to use this function)
    Select Release Seats and then click the Cancel Order button
    Close the Order Details tab in your browser and return to the Event Detail Report tab in your browser
    Repeat this process for each order until all orders have been cancelled
  2. Issue a credit to the ticket buyer through your payment gateway
    Login to your payment gateway and issue a credit (refund) to each customer. You can print out the Event Detail Report used previously and lookup each order in your payment gateway by last name, order number, or transaction date/time.
