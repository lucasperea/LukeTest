Post Installation Configuration

Margins
We recommend setting the browser's margins to .25 See the instructions below to set your margins within your browser.
Note: These are recommendations, you may wish to adjust more to satisfy the desired ticket layout

Microsoft Internet Explorer Mozilla FireFox Goole Chrome
  1. Open Internet Explorer
  2. Press the Alt key to get the drop down menus and click File.
  3. Click Page Setup.
  4. Set all Margins to .25
  5. click OK
  1. Press the Alt key to get the drop down menus and click File.
  2. Click File
  3. Click Page Setup
  4. Click Margins & Headers/Footers tab
  5. Change all margins to .25
  1. Click Print
  2. Within the Print Dialog Box click More Settings
  3. Select "Minimum" from the Margin options

Header and Footer
We strongly recommend setting the "Header/Footer" of your browser to empty or blank. See the instructions below to remove Header/Footer within your browser.
Microsoft Internet Explorer Mozilla FireFox Goole Chrome
  1. Open Internet Explorer
  2. Press the Alt key to get the drop down menus and click File.
  3. Click Page Setup.
  4. Remove "&w&bPage &p of &P" from the header if you want to not have a header printed and "&u&b&d" from the footer if you do not want to have a footer printed.
  1. Press the Alt key to get the drop down menus and click File.
  2. Click File
  3. Click Page Setup
  4. Click Margins & Headers/Footers tab
  5. Change the top three and bottom three drop down menus to "--blank--" if you want to remove all the headers and footers.
  1. Click Print
  2. Within the Print Dialog Box click More Settings
  3. Uncheck "Headers and Footers"
